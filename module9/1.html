What is a file?

File is a named location on disk to store related information. it's wont to permanently store data during a non-volatile memory (e.g. hard disk).

Since, random access memory (RAM) is volatile which loses its data when the pc is turned off, we use files for future use of the info .

When we want to read from or write to a file we'd like to open it first. a file operation takes place within the following order.

  1. Open a file

    Python has a built-in function open() to open a file. This function returns a file object, also called a handle, as it is used to read or modify the file accordingly.

    >>> f = open("test.txt")    # open file in current directory

    >>> f = open("C:/Python33/README.txt")  # specifying full path


    We can specify the mode while opening a file. In mode, we specify whether we want to read 'r', write 'w' or append 'a' to the file. We also specify if we want to open the file in text mode or binary mode.

  2.  Read or write (perform operation)

    To read a file in Python, we must open the file in reading mode.

    There are various methods available for this purpose. We can use the read(size) method to read the size of data. If the size parameter is not specified, it reads and returns up to the end of the file.

    >>> f = open("test.txt",'r',encoding = 'utf-8')

    >>> f.read(4)    # read the first 4 data

    'This'

     

    >>> f.read(4)    # read the next 4 data

    ' is '

     

    >>> f.read()     # read in the rest till end of file

    'my first file\nThis file\ncontains three lines\n'

    3.Close the file

When done with operations, there is a need to properly close this file. a file when closed,  frees up the resources that were tied with it. This is achieved using Python close() method.

Python does have a garbage collector to clean up unreferenced objects but, we must not rely on it to close the file.
