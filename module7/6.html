Anonymous Functions in Python

Anonymous function is a function that is defined without a name.

Anonymous functions also are called lambda functions in Python because rather than declaring them with the quality def keyword, you employ the lambda keyword




doub = lambda x: x**2


x is the argument, and x*2 is the expression or instruction that gets evaluated and returned.

The filter() function in Python takes in a function and a list as arguments.

The function is called with all the items in the list and a new list is returned which contains items for which the function evaluates to True.

The map() function returns a map object(which is an iterator) of the results after applying the given function to each item of a given iterable (list, tuple etc.)

The zip() function is a function that takes a number of iterables and then creates a tuple containing each of the elements in the iterables. Like map(), it returns a generator object, which can be easily converted to a list by calling the built-in list function on it.
