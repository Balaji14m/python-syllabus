Introduction to Function

<video introduction to function>

In Python, a function is a group of related statements that perform a selected task.

Functions help break our program into smaller and modular chunks. As our program grows larger and bigger , functions make it more organized and manageable.

Furthermore, it avoids repetition and makes code reusable.

Types of function:

There are three sorts of functions in Python:

  1. Built-in functions, like help() , min() to find the minimum value, print() to print an object to the terminal etc.

  2. User-Defined Functions (UDFs), which are functions that users create to assist them out;

  3. Anonymous functions, which also are called lambda functions because they're not declared with the quality def keyword.
