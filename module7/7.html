Types of Functions (recursive and anonymous functions)

Anonymous Functions in Python

Anonymous function is a function that is defined without a name.

Anonymous functions also are called lambda functions in Python because rather than declaring them with the quality def keyword, you employ the lambda keyword




doub = lambda x: x**2


x is the argument, and x*2 is the expression or instruction that gets evaluated and returned.
