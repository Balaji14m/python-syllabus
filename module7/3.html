Types of return statements

Types of return statement:

If we want the functions to work as expected, there must be a return statement in every programming language. The return keyword is to exit a function and return a value.

The function is a series of statements that returns some value to a caller. It can also be passed by zero or more parameters that may be used in the execution of the body.

The return is a built-in Python statement or keyword used to end the execution of a function call and �returns� the result (value of the expression following the return keyword) .

The statement after the return statement is not executed. If the return statement is without any expression, then �None� is returned.

Note: return statement in Python can not be used outside the function.
