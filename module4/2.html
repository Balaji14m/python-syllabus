Indentation and Related Errors


Mostly in other programming languages, a block of codes for class and functions or control structure/flow is indicated using braces but in python, there is no need to provide any braces. Here, a block of code is denoted by indentation of the line which is imposed forcefully by python. Whitespace is known as indentation in Python.

So if you do not give an indentation for the block of codes python will give an error.
