Understanding modules

 Modules:

When we are coding on the Python interpreter and quit it ;enter it again, the function and variable that we have defined before are lost.When we want to write the longer program we write the code by using the script. When the  program gets longer, we may split it into several files for easier maintenance. Also sometimes we want to use a function that is already written in several programs without copying its definition into each program.

To implement such a thing python allows to put definitions in a file and use them in a script or in an interactive instance of the interpreter. Such a file is called a Module; definitions of the module can be imported into other modules or into the main module.

Module is the file containing definition and statement. We can define the operation in the file and save with .py and with the help of import statement we can use it in another file.

For eg:

1.Create the module file where add the operation of addition in function:

2.Save the file with add.py and run the file

3.Import the module with the import statement and print the addition.
