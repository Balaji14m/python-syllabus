Using Python Interpreter (Interactive Mode)


For making programming easy and interactive Python provides two basic modes: a script mode  and an interactive. A script mode is the normal mode of python where the scripted and completed .py files are run in the python interpreter.On the other hand an interactive mode is a command line shell which gives immediate feedback for each statement, while running previously fed statements in active memory. As new lines are fed into the interpreter, the fed program is evaluated both in part and in whole.
