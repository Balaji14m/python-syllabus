Polymorphism


Polymorphism means multiple forms. for instance Eating is a process performed by all living things but food and way of eating is different.

This is called polymorphism.


Example :- 

class A():


    def __init__(self):

        print("Constructor of class A called....!")

    def msg(self):

        super().msg()

        print("Method of Class A msg")


class B():


    def __init__(self):

        print("Constructor of class B called....!")

    def msg(self):

        print("Method of Class B msg")


class C(A,B):


    def __init__(self):

        print("Constructor of class C called ....!")


    def msg(self):

        super().msg()

        print("Method of class C msg")


def main():

    c=C()

    c.msg()

    c.msg()

    c.msg()


if __name__=='__main__':

    main()

#polymorphism 2


class India:


    def getCapital(self):

        print("Delhi")

    def getCurrency(self):

        print("INR Rs.")


class US:


    def getCapital(self):

        print("Washington Dc")


    def getCurrency(self):

        print("US $")


def main():


    i =India()

    us=US()

    for a in (i,us):

        a.getCapital()

        a.getCurrency()

 in python overriding is possible but Overloading not possible 

Because method overloading, that is, it is not possible to define more than one method with the same name in a class in python. This is because method arguments in python do not have a type. A method accepting one argument can be called with an integer value, a string or a double.


We can perform method overloading by giving default argument or variable length argument in python.
