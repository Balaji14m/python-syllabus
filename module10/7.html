Operator Overloading in python


Python provides functionality to overload operators.

Operator Overloading means giving extended meaning beyond their predefined operational meaning. for instance operator + is employed to feature two integers also as join two strings and merge two lists. it's achievable because �+� operator is overloaded by int class and str class. you would possibly have noticed that an equivalent built-in operator or function shows different behavior for objects of various classes, this is often called Operator Overloading.
