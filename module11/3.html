Handling an Exception (try, except, finally, else)

Try -except Block in Exception:

try:

    Code likely to have an error

except XXX:

      Alternative Code


  • finally block:

The main purpose of finally block is to maintain clean up code.

try:

    Code likely to have error

except:

     Alternative Code

finally:

     Cleanup code


The specialty of final block will be executed always whether exception raised or not raised and whether exception handled or not handled.Whenever we are using os._exit(0) function then Python Virtual Machine itself will be shutdown.In this particular case finally won't be executed.

  • else block with try-except-finally:

We can use else block with try-except-finally blocks.else block will be executed if and only if there are no exceptions inside try block.

Try:

    Code likely to have error

except:

    will be executed if exception inside try

else:

     will be executed if there is no exception inside try

finally:

      will be executed whether an exception raised or not       raised and handled or not handled
