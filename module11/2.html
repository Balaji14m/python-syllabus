Types of in-built Exceptions

Syntax Errors:

The errors which occur because of invalid syntaxes are often called syntax errors.

Example :

print "Welcome"

SyntaxError: Missing parentheses in call to 'print'


7.2 Runtime Errors:

These are also known as exceptions. While executing the program if something goes wrong because of end-user input or programming logic or memory problems etc then can get Runtime Errors.

Example :

print(5/0) ==>ZeroDivisionError: division by zero

print(5/"ten") ==>TypeError: unsupported operand type(s) for /: 'int' and 'str'
